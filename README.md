# BukavuWasteCollectedDataAnalysis
This repository contains the complete source code used for the statistical analysis of waste collection data in the city of Bukavu (Democratic Republic of Congo). The data were collected by seven EM310-UDL ultrasonic sensors communicating via the LoRaWAN protocol, deployed for 12 weeks in three types of neighborhoods (residential, dense, and peri-urban). The code covers all the analyses performed: data preprocessing, descriptive analysis of waste production, user behavior analysis, monitoring of emptying events, bin fill time, predictive modeling (random forests), sensor clustering using K-means, principal component analysis (PCA), and projection of waste production in the city of Bukavu to 2030.
