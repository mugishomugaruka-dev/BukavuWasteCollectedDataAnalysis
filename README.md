# BukavuWasteCollectedDataAnalysis
This repository contains the complete source codes used for the statistical analysis of waste collection data in the city of Bukavu (Democratic Republic of Congo). The data were collected by seven EM310-UDL ultrasonic sensors communicating through the LoRaWAN protocol, deployed for 12 weeks in three types of neighborhoods (residential, dense, peri-urban). The code covers all the analyses performed: data loading and concatenation, cleaning, missing value imputation using optimized KNN, descriptive and inferential analyses (t-test, ANOVA, chi-square), predictive modeling (linear regression, logistic regression, random forests, Prophet), clustering (K-Means, PCA), anomaly detection (Isolation Forest), and weighted extrapolation to the scale of the city of Bukavu.
